# Pole Vault Site

## What changed

- Resume section removed.
- Professional content combined into one main page:
  - `html/index.html`
- Scratch page and app still separate:
  - `html/scratch.html`
  - `html/app.html`

## How to open

1. Open `html/index.html`.
2. Open `html/scratch.html`.
3. Open `html/app.html`.

## GitHub Pages

1. Push the `course-site` folder to a GitHub repo.
2. In GitHub: `Settings` -> `Pages`.
3. Set Source to `Deploy from a branch`, then choose `main` and `/ (root)`.
4. Use this URL:
   - `https://<your-username>.github.io/<repo-name>/html/index.html`
